# 📘 Projet 1 — Prenez en main votre formation Data Analyst  
*OpenClassrooms – Portfolio Data Analyst*

## 🎯 Objectif du projet  
Ce premier projet a pour but de découvrir le fonctionnement de la formation Data Analyst, d’identifier les compétences attendues, de comprendre les modalités d’accompagnement et de commencer à structurer son organisation personnelle.

Il s’agit d’un projet d’introduction, mais essentiel : il pose les bases de la posture professionnelle du futur Data Analyst.
